Voir quel process est actif sur un port spécifique ainsi que le numéro de PID :
netstat -aon | findstr :<Numéro de port>

Afficher les régles firewall d'un port sur le pc actuel :
netsh advfirewall firewall show rule name=all | findstr <Numéro de port>

Ajouter/Retirer un régle firewall sur un port du pc actuel (qui autorise la com TCP sur ce port, côté PC) :
netsh advfirewall firewall add rule name="Port 3333" dir=in action=allow protocol=TCP localport=<Numéro de port>
netsh advfirewall firewall remove  rule name="<Nom de la régle>"

Pour generer un executable avec pkg, utiliser la dernière version de node est ok mais il est important d'utiliser node16 dans la commande.
